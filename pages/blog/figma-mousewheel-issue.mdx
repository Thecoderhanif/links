## If you a Greasemonkey user, try this one!

Bagi pengguna Greasemonkey, mungkin lebih berguna jika menggunakan script ini.

```js:figma-mouse-wheel.user.js
const MULTIPLIER_X = 30;
const MULTIPLIER_Y = 10;

function speedUp(ev) {
  if (!ev.isTrusted) return true;

  const { clientX, clientY, screenX, screenY, shiftKey, ctrlKey, altKey } = ev;
  const newEv = new WheelEvent('wheel', {
    bubbles: true,
    cancelable: true,
    view: window,
    clientX,
    clientY,
    screenX,
    screenY,
    ctrlKey,
    shiftKey,
    altKey,
    deltaX: ev.deltaX * MULTIPLIER_X,
    deltaY: ev.deltaY * MULTIPLIER_Y
  });

  ev.preventDefault();
  ev.stopPropagation();
  ev.target.dispatchEvent(newEv);
  return false;
}

function addCanvasEvent() {
  const canvas = document.querySelector('.view > canvas');
  if (canvas != null) canvas.addEventListener('wheel', speedUp);
  else setTimeout(addCanvasEvent, 500);
}
addCanvasEvent();
```

## Poor man with complicated solution

_"Kalo ada yang sulit kenapa harus yang ~~sulit~~ mudah"_. Hmm, gak sedikit emang manusia yang memiliki sifat seperti ini

Oke. Jadi gini kira-kira caranya, ya sebenernya gak terlalu sulit juga sih.

1. Buka `about:config`
2. Lalu masukan keyword

<!-- ```bash
mousewheel.with_control.delta_multiplier_y
``` -->

3. Ubah value dari konfigurasi tersebut, disini saya mengubahnya menjadi 1000
4. And that's it!

Sayangnya cara ini cuma mengatasi masalah **Ctrl+Mousewheel** (Zoom in/out)
