> Hello World

Sama halnya ketika memulai belajar bahasa pemrograman, ekspresi ini mungkin sangat familiar. Setiap kali belajar bahasa baru. Saya, mungkin juga Kamu pasti memulainya dengan membuat program sederhana yang tidak begitu berguna hanya dengan menampilkan kata _Hello World_ pada Konsol atau Peramban. That's good!, setidaknya itu berarti Kita mulai mempelajari sesuatu.

Dan tulisan ini—Hello World! Yet Again—merupakan tulisan pertama pada Blog ini dengan domain kepemilikan pribadi tentunya.

Sebagai pengenalan, Blog ini dibangun dengan menggunakan beberapa teknologi yang memudahkan saya dalam proses pengembangan, diantaranya:

- Next.js — sebagai frontend framework
- Chakra UI — untuk styling (Component Library)
- Firebase — Hey, jangan sungakan untuk keluarkan pendapatmu di [Halaman ini](/advices)
- MDX — konten majamenen untuk orang ribet(?)

Bentar, mau coba test `kodingan` dulu

```jsx:pages/foo.jsx
const Foo = () => {
  return (
    <div>
      <p>Foo bar</p>
    </div>
  );
};
```
