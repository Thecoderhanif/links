Asus X455L, yang mana merupakan laptop pemberian, terima kasih untuk Bu Kiki yang sudah menghibahkan kepada saya.

Dengan kondisi awal processor Intel® Core™ i3 Gen4 + RAM DDR3L 2GB onboard cukup frustasi untuk menjalankan Sistem Operasi [KDE Neon](https://neon.kde.org/) yang digunakan saat itu. Dengan menambah RAM 8GB dan menggunakan [Adata SU800 SSD 256GB](https://www.adata.com/us/specification/410) saya rasa sudah sangat jauh lebih baik dengan performa yang diberikan untuk kebutuhan Web Development.

GNU/Linux

**Dotfiles?** tentu saya menyimpan setiap konfigurasi yang saya gunakan di [repositori github](https://github.com/opxop/dotfiles/).

(masih) Visual Studio Code! dan VIM kadang-kadang

Sampai sekarang saya masih dibuat nyaman oleh VS Code untuk menulis kode. Ekstensi yang tersedia membantu sekali dalam produktivitas penulisan kode. Saya menyediakan beberapa ekstensi yang saya gunakan dan menyimpannya di [Gist](https://gist.github.com/opxop/b1fc4207f2b622c761d1aceeb4191939).

Juga memberikan hari khusus untuk berusaha mengurangi pengunaan mouse dengan menerapkan [No Mouse Day](https://www.a11yproject.com/posts/2020-10-15-no-mouse-days/). Memang alasan yang cukup tidak masuk akal, tapi ini berhasil untuk mengingat kembali shortcut-shortcut yang cepat saya lupakan, tidak hanya pada [VIM](https://neovim.io/), tapi keseluruhan penggunaan ekosistem desktop.

### Misc

- Terminal emulator menggunakan [Kitty](https://sw.kovidgoyal.net/kitty/) karena mendukung multiple tab dan [st](https://st.suckless.org/) untuk merasakan program suckless.
- Peramban utama menggunakan [Firefox](https://www.mozilla.org/en-US/firefox/developer/), selain itu menggunakan [Brave](https://brave.com/) untuk beberapa alasan tertentu.
- Note-taking, todo list, dsb menggunakan [Notion](https://www.notion.so/) terlepas dari saya seorang yang produktif atau tidak.
- Password manager menggunakan [Bitwarden](https://bitwarden.com/) dan [Authy](https://authy.com/) untuk mengelola 2FA.
- Bookmark manager menggunakan [Raindrop](https://raindrop.io/).
- Membuat antarmuka dan desain menggunakan [Figma](https://www.figma.com/).
- Untuk mendengarkan lagu saya menggunakan [Spotify](https://spotify.com/) yang memiliki total kalatog lagu terbanyak (katanya), karena beberapa lagu dalam playlist saya pun hanya tersedia di platform ini.
